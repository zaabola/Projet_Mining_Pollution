How It Works

This module uses a YOLOv8 model trained on wildlife data to monitor animal populations and assess mining impact on habitats:

  1. Single Image Detection — upload any image to detect and count animals present
  2. Habitat Comparison — upload before/after images of the same area to measure population change
  3. Mining Safety Check — if animals are detected, a warning is issued: "DO NOT MINE HERE"
Tip: Use the Before/After comparison to check if mining operations have displaced wildlife from an area. If any animals are detected, the area should not be mined.
